# Roadmap

## 0. Framework

- Monorepo with core, RAG, agent, CLI, API, Web, and MCP skeletons.
- Zi Wei Dou Shu domain model.
- Experimental chart scaffold for demos.
- Source policy for private copyrighted materials.

## 1. Calendar Foundation

- Solar to lunar conversion provider.
- Timezone and true solar time options.
- Jieqi provider.
- Golden fixtures for known dates.

## 2. Zi Wei Dou Shu Engine

- Life palace and body palace.
- Five-element bureau.
- Zi Wei and Tian Fu star systems.
- Major stars and auxiliary stars.
- Four transformations.
- Decade, annual, monthly, and daily flows.

## 3. RAG Knowledge Layer

- Local text ingestion.
- Metadata-first source registry.
- Topic taxonomy: palace, star, transformation, pattern, flow, school.
- Hybrid retrieval adapter.
- Citation-grounded report schema.

## 4. Agent Workbench

- Tool-calling planner.
- Report composer.
- Contradiction and unsupported-claim checker.
- Multi-school comparison mode.
- Web UI and shareable static report.

## 5. Developer Ecosystem

- NPM package.
- MCP server.
- REST API.
- Example notebooks and fixtures.
- Public demo with synthetic or licensed corpus.
